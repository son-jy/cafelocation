# 카페 창업을 위한 서울시 상권 입지 분석
- 기간 : 2024.9.9 ~ 2024.10.18
- 인원 : 4명
- 기술 : Python, Jupyter, Pandas, scikit-learn, numpy, matplotlib, seaborn, Folium

## 프로젝트 개요
국내 외식업 중 카페 창업은 진입 장벽이 낮아 2030 세대의 도전이 활발하지만, 점포 수가 치킨집을 추월할 정도로 시장 포화와 무한 경쟁이 심화된 상황입니다.
본 프로젝트에서는 서울 열린데이터광장의 상권별 유동인구, 상주인구, 직장인구 및 매출액 데이터와 부동산 임대료 데이터를 결합하여 서울시 상권의 다차원적 특성을 분석하였습니다.
이를 통해 보편적인 기준에서 벗어나, 예비 창업자 개개인의 컨셉과 영업 방향성에 맞춤화된 최적의 개업 입지를 추천하는 데이터 기반 비즈니스 모델 제안을 목표로 합니다.

## 프로젝트 목표
- **상권 다차원 데이터셋 구축:** 서울시 상권 데이터와 임대료 데이터를 매칭 및 정제하여 다각적 분석 프레임워크 구축
- **K-means++ 기반 상권 세그멘테이션:** 인구통계학적 및 배후지 특성 변수를 활용해 서울시 상권을 고유 특징을 가진 4개 그룹으로 재분류
- **창업 컨셉별 맞춤형 매칭 모델 정립:** 가상 창업자 프로파일(A~D 타입)을 정의하고 그룹별 소비·인구 특성 스코어링을 통한 최적의 상권 매칭
- **잠재 매출 상위 상권 추천 및 시각화:** 매칭된 그룹 내 매출 잠재력이 가장 높은 상권을 선별하고, Folium을 활용하여 직관적인 지도 시각화 구현

## 데이터 출처
- 상권 기반 데이터: 서울 열린데이터광장 '서울시 상권분석서비스' 공공 데이터셋 (점포, 직장인구, 유동인구, 상주인구, 소득소비, 추정매출, 집객시설 등)
- 임대료 데이터: 한국부동산원 상업용부동산 분기별 지역별 임대료 데이터

## 분석 결과
서울시 공공 데이터와 임대료 데이터를 결합하여 상권을 4개 유형으로 재분류하고, 창업 컨셉별 최적의 입지를 매칭했습니다.
- **그룹 1. 주거 밀집 상권 (애놀라홈즈)**
    - 데이터 특성: 아파트 단지 수 및 총 상주인구 수가 서울시 평균을 크게 상회.
    - 비즈니스 매칭: 고객 유대감과 재방문율이 높은 **'단골 고객 중심의 동네 편안한 카페'** 컨셉에 최적.
- **그룹2. 대학가 상권 (젠리단길)**
    - 데이터 특성: 연령대별 점주 중 2030 비율이 높고, 전체 유동인구 중 2030 비중이 약 40%를 차지하나 월평균 소득은 가장 낮음.
    - 비즈니스 매칭: 카공족 및 학기 중 정기적인 유동인구를 타겟으로 하는 **'안정적인 프랜차이즈 카페'** 창업에 최적.
- **그룹3. 오피스 타운 상권 (커핏팡잇츠)**
    - 데이터 특성: 총 직장인구 수 1위, 주중 유동인구 대비 매출액 효율이 가장 압도적인 비즈니스 밀집 지역.
    - 비즈니스 매칭: 테이크아웃 위주로 피크타임(출근·점심)이 명확하고 빠른 퇴근이 가능한 **'소형/저가 커피 전문점'** 컨셉에 최적.
- **그룹4. 상업 활성화 상권 (트·민·상)**
    - 데이터 특성: 전체 유동인구 1위, 야간(21~24시) 매출 비중 최상위, 신생 점포 진입과 교체가 활발한 다이나믹한 번화가.
    - 비즈니스 매칭: 트렌디한 MZ세대의 소비가 집중되는 **'독특한 인테리어 및 시그니처 디저트 중심의 개인 카페'** 컨셉에 최적.
